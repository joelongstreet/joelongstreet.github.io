FartCharade

so dumb

When Twitter announced their streaming API in 2012, I really really wanted to find a reason to use it. An auto updating realtime sidebar social widget sounded excruciatingly uninteresting. But don’t worry, it didn’t take long for genius to strike.

Introducing FartCharade. A real time Twitter client which loudly and offensively notifies viewers of social media's obsession with farting. It’s funny.

But srsly, creating this app was a tremendous learning experience. It was one of the first projects I wrote in Node, it introduced me to streaming apis, helped me practice web sockets, and gave me a great excuse to try out Bootstrap.

Unfortunately, the venture money fell through for FartCharade and no further development is planned on this project. Github Repo.
